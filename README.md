# shooter
Этот проект создается в рамках курса cpp. 
Авторы: Гондаренко Анастасия и Стрельникова Валентина Б04-402. 
Общая идея: мы создаем игру по типу стрелялки, но с "ванильным" интерфейсом. Стрельба производится не пулями, а конфетками по злым животным. Когда конфета попадает в животное оно добреет и пеерстает быть опасным
